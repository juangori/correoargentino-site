# Preguntas Frecuentes

### Que servicios de Correo Argentino soporta?

Soporta MiCorreo (para PyMEs y monotributistas, con cotización) y Paq.Ar v2 (para empresas con acuerdo comercial, con etiquetas). [Leer más →](https://correoargentinopro.com/blog/que-es-micorreo-correo-argentino.html)

### Necesito tener cuenta en Correo Argentino?

Si. Necesitas una cuenta en MiCorreo o un acuerdo comercial Paq.Ar. El plugin se conecta a tu cuenta existente.

### La licencia es por sitio?

Si, cada licencia anual es valida para 1 dominio. Podes desactivarla y moverla a otro sitio cuándo quieras.

### Cuanto dura la licencia?

La licencia tiene una vigencia de 1 año. Incluye actualizaciones y soporte durante ese periodo. Al vencer, podes renovarla para seguir recibiendo actualizaciones y soporte.

### Que pasa cuándo vence la licencia?

El plugin deja de funcionar (cotizar, crear envíos, generar etiquetas). Desde el panel del plugin vas a ver un aviso con el link para renovarla.

### Incluye actualizaciones?

Si, mientras tu licencia este activa. Las actualizaciones llegan directamente a tu WordPress, como cualquier plugin del repositorio oficial.

### Es compatible con mi tema?

El plugin funciona con cualquier tema compatible con WooCommerce. Soporta HPOS y el checkout por bloques. [Ver guía de integración →](https://correoargentinopro.com/blog/como-integrar-correo-argentino-woocommerce.html)

### Como recibo la licencia?

Despues de pagar con Mercado Pago, tu clave de licencia aparece en pantalla y se envia a tu email. La activacion es instantanea.

### Puedo usar el plugin en mas de un sitio?

Cada licencia es para 1 sitio. Si necesitas usarlo en multiples sitios, podes comprar licencias adicionales.

### Que pasa si cambio de dominio?

Podes desactivar la licencia en el sitio actual y activarla en el nuevo dominio desde el panel del plugin.

### Funciona con el checkout por bloques?

Si. El plugin es totalmente compatible con el nuevo checkout por bloques de WooCommerce, ademas del checkout clásico. [Ver guía WooCommerce →](https://correoargentinopro.com/blog/woocommerce-argentina-guia-completa.html)

### Que version de WooCommerce necesito?

Se requiere WooCommerce 7.0 o superior. Recomendamos siempre usar la ultima version disponible.
